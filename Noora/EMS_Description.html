Introduction

This project is a model of an employee management system. It contains all details about company's employees and projects they work on. It has all the data about their education, degrees, bank account, contacts, etc.


Methodology

The following technologies were used to build this project satisfying the stated requirements:

  • XML
  • XML Schema
  • HTML
  • XSL
  • XSLT

Work content

The whole project conssits of several files:

XML File
Main file with project data
XML Schema File
A file describing the XML structure
XSL Transforamtion File
Transforamtion to an HTML table
HTML File
Final result of XSLT

Elements

The root element in the project is <system>. As required it contatins 5 records of <employee>.

Each recors of <employee> has:

Basic details about employees

  • <firstName>
  • <lastName>
  • <age>
  • etc.

Contact information

  • <phone>
  • <email>
  • <address>
  • etc.

Job information

  • <position>
  • <team>
  • <department>
  • etc.

Salary details

  • <amount>
  • <bankAccount>
  • <bankName>
  • etc.

Education

  • <institute>
  • <degree>
  • <spesialisation>
  • etc.

Transformation

Using XSLT the XML doucment was transformed to an HTML table contating information about the employees.

Sorted by lastname, employees with salaries greater than 15000 are highlighted by green color, employees with working hours greater than 50 are highlighted with yellow.

As follows

In order to create such table after transformation <xsl:for-each>, <xsl:sort> and <xsl:choose> have been used.


Conclusion

This project is a representation of an employee management system, with some extra features it can be used to implement a real solution which can be integrated to almost any kind of company.
